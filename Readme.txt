USB FORWARDING BY RI - QUICK START GUIDE
INSTALLATION & USAGE:

Double-click USB Forwarding By RI.exe to automatically install the setup and drivers.

Open the software, enter the Phone's IP address, and click "Attach" for your USB    device.

TROUBLESHOOTING:
If the application fails to attach a device, please verify that the virtual driver was installed correctly:

Open Windows "Device Manager".

Scroll down and expand "Universal Serial Bus controllers".

Verify that "usbip-win VHCI(ude)" is listed.

If the driver is missing, simply delete the desktop shortcut and run the main USB Forwarding By RI.exe file again to trigger a fresh installation.
